# 🧠 OCaml Machine Learning

> A small collection of machine learning examples in OCaml

## Current Entry Points

- `xor` trains a tiny neural network to learn XOR.
- `gpt` is a placeholder for future GPT examples.
- `transformers` is a placeholder for future transformer examples.

## Build

```sh
dune build
```

## Run the XOR Example

```sh
dune exec ./bin/xor.exe
```

## Run the Backend Benchmark

```sh
dune exec ./benchmark/xorBackendBenchmark.exe
```

## Run Tests

```sh
dune runtest
```
